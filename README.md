# Libre Video SSR Proxy

SSR proxy between API and Frontend of the Libre Video web application.

## Used Tools

- PassportJS.
- Helmet.
- Express.
- Axios.

## Routes

- POST /auth/sign-in
- POST /auth/sign-up
- GET /auth/google-oauth
- GET /auth/google
- GET /auth/twitter
- GET /auth/linkedin
- GET /auth/facebook
- GET /movies
- GET /user-movies
- POST /user-movies
- DELETE /user-movies/:userMovieId
